# hpe-software-engineering-sim
Backend architecture and system optimization tasks for the HPE virtual experience program.
I recently participated in Hewlett Packard Enterprise’s Software Engineering job simulation on Forage. I got a real feel for what it might be like to work as a back end engineer by building a web server application in Java Spring Boot and developing unit tests to assess its performance.

Having the opportunity to work on a web server application from end to end helped me to realize that I really enjoy working as a back end engineer and love seeing a project through to completion. I’d love to apply what I’ve learned during this simulation in a software engineering team at Hewlett Packard Enterprise.
